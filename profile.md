## 💼 Key Projects & Experience

### Zero-Secret EKS Platform | *Cloud / DevOps Engineer*
*A production-ready Kubernetes playground built to completely eliminate static passwords.*
*   **The Impact:** Got developer onboarding down to under 5 minutes and achieved 100% IaC coverage with absolutely zero static secrets.
*   **Security First:** Built a "Zero-Secret" CI pipeline using GitHub OIDC and EKS Pod Identity. This means we map IAM roles straight to ServiceAccounts, completely avoiding the nightmare of leaked long-lived access keys.
*   **GitOps Delivery:** Set up a multi-tenant, pull-based GitOps flow using ArgoCD. I also built standardized Helm templates to keep staging and production environments cleanly separated and drift-proof.
*   **Infrastructure:** Managed the whole AWS setup (VPCs, NAT gateways, EKS control plane) using Terraform, and automated load balancing and SSL certificates so nobody has to do it manually.

### Plush Haven E-Commerce Store | *Cloud / DevOps Engineer*
*A highly available retail platform featuring a fully containerized microservices setup.*
*   **The Impact:** Made the platform capable of automatically scaling out in just 60 seconds when traffic spikes, maintaining 100% IaC across 6 different microservices.
*   **Infrastructure Design:** Designed a microservices architecture (a frontend and 5 backends) on AWS ECS Fargate, sitting safely behind an Application Load Balancer. Data lives securely in an RDS PostgreSQL database.
*   **Smooth CI/CD:** Set up dedicated deployment pipelines using GitHub Actions. We have automated rolling updates for staging, and manual dispatch gates so we're always in control of what goes to production.
*   **Observability:** You can't fix what you can't see! I added a full monitoring stack with Prometheus, Grafana, and Loki. I also ran k6 load tests to make sure the platform can handle real-world traffic.
